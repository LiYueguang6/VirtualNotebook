# Java容器
***
容器主要有两种Collection和Map
* Collection包括Set、List、Queue
* Map包括TreeMap、HashMap、HashTable（类似HashMap，线程安全）、LinkedHashMap
## 设计模式

##### 迭代器模式
Collection继承了Iterable接口。（所有继承了Iterable接口的类，都可以使用 **iterator()** 方法产生Iterator对象来遍历；JDK1.5之后可以使用 **foreach** 方法遍历实现Iterable接口的对象）
##### 适配器模式（这里不是很懂）
```java
java.util.Arrays
```
asList()可以把数组转化为List类型
